# 核心特性

Meblog-Java提供了丰富的核心功能，涵盖了博客系统所需的各个方面。以下是对主要核心特性的详细介绍：

## 1. 文章管理系统

### 1.1 功能概述

文章管理是博客系统的核心功能，Meblog-Java提供了完整的文章生命周期管理，包括文章的创建、编辑、发布、删除、草稿保存等功能。

### 1.2 主要功能点

- **文章创建与编辑**：支持Markdown格式，提供丰富的编辑功能
- **草稿管理**：支持文章草稿的保存和恢复
- **文章发布**：支持定时发布和立即发布
- **文章状态管理**：支持已发布、草稿、回收站等多种状态
- **文章分类与标签**：支持为文章添加分类和标签
- **文章评论**：支持文章评论功能（预留接口）
- **文章阅读量统计**：自动统计文章阅读量
- **文章点赞功能**：支持文章点赞功能（预留接口）

### 1.3 技术实现

- 采用Markdown格式存储文章内容
- 支持HTML转换和预览
- 实现了文章的版本管理
- 采用Redis缓存热门文章

### 1.4 使用场景

- 博主发布技术文章
- 企业发布内部文档
- 内容创作者发布各类文章

## 2. 分类与标签管理

### 2.1 功能概述

分类与标签是组织和管理文章的重要方式，Meblog-Java提供了灵活的分类和标签管理系统。

### 2.2 主要功能点

- **分类管理**：支持分类的创建、编辑、删除和排序
- **多级分类**：支持无限级分类嵌套
- **标签管理**：支持标签的创建、编辑、删除
- **标签云**：支持标签云的生成和展示
- **分类统计**：自动统计每个分类下的文章数量
- **标签统计**：自动统计每个标签下的文章数量

### 2.3 技术实现

- 分类采用树形结构设计
- 标签采用多对多关联设计
- 实现了分类和标签的缓存机制

### 2.4 使用场景

- 按技术领域组织技术文章
- 按业务模块组织企业文档
- 按主题分类各类内容

## 3. 全文搜索功能

### 3.1 功能概述

全文搜索是博客系统的重要功能，Meblog-Java基于Lucene实现了高效的全文搜索功能。

### 3.2 主要功能点

- **多维度搜索**：支持文章标题、内容、分类、标签等多维度搜索
- **关键词高亮**：搜索结果支持关键词高亮显示
- **搜索排序**：支持按相关性、发布时间等排序
- **搜索建议**：支持搜索关键词建议（预留功能）

### 3.3 技术实现

- 基于Lucene 8.x实现
- 实现了索引的自动更新和优化
- 支持搜索结果的分页和缓存

### 3.4 使用场景

- 用户查找特定主题的文章
- 开发者查找相关技术文档
- 管理员查找特定内容

## 4. 权限管理系统

### 4.1 功能概述

权限管理是保证系统安全的重要组成部分，Meblog-Java基于JWT和Spring Security实现了完整的权限管理系统。

### 4.2 主要功能点

- **用户管理**：支持用户的创建、编辑、删除和状态管理
- **角色管理**：支持角色的创建、编辑、删除和权限分配
- **权限管理**：支持细粒度的权限控制
- **JWT认证**：基于JWT的无状态认证
- **密码加密**：采用BCrypt加密算法存储密码
- **登录日志**：记录用户登录日志（预留功能）

### 4.3 技术实现

- 基于Spring Security 5.x实现
- 采用JWT进行身份认证
- 实现了RBAC（基于角色的访问控制）模型
- 支持接口级别的权限控制

### 4.4 使用场景

- 管理员管理后台
- 不同角色用户访问不同资源
- 保护敏感接口和数据

## 5. 文件存储支持

### 5.1 功能概述

文件存储是博客系统的重要功能，Meblog-Java集成了Minio对象存储，支持图片、文件的上传、存储和访问管理。

### 5.2 主要功能点

- **文件上传**：支持多种文件类型的上传
- **文件管理**：支持文件的查看、删除和下载
- **图片处理**：支持图片的缩放、裁剪等处理（预留功能）
- **文件访问控制**：支持文件的访问权限控制
- **文件存储策略**：支持多种存储策略配置

### 5.3 技术实现

- 集成Minio对象存储
- 支持本地文件存储作为备选
- 实现了文件上传的分片上传和断点续传（预留功能）
- 采用CDN加速文件访问（预留配置）

### 5.4 使用场景

- 文章中插入图片
- 上传和分享文档文件
- 存储网站静态资源

## 6. 统计分析功能

### 6.1 功能概述

统计分析功能帮助网站管理员了解网站的运营情况，Meblog-Java提供了丰富的统计分析功能。

### 6.2 主要功能点

- **文章统计**：统计文章总数、发布数量、草稿数量等
- **访问统计**：统计网站访问量、独立访客数等
- **阅读量统计**：统计文章阅读量、热门文章等
- **用户统计**：统计用户数量、活跃用户等
- **分类统计**：统计各分类下的文章数量
- **标签统计**：统计各标签下的文章数量

### 6.3 技术实现

- 采用Redis统计实时数据
- 实现了定时任务同步统计数据到数据库
- 支持数据可视化展示（前端实现）

### 6.4 使用场景

- 网站管理员了解网站运营情况
- 分析热门文章和内容
- 优化网站内容策略

## 7. Wiki文档系统

### 7.1 功能概述

Wiki文档系统是Meblog-Java的一个重要扩展功能，支持知识库的创建和管理。

### 7.2 主要功能点

- **Wiki页面管理**：支持Wiki页面的创建、编辑、删除
- **Wiki分类**：支持Wiki页面的分类管理
- **Wiki版本**：支持Wiki页面的版本控制
- **Wiki搜索**：支持Wiki内容的全文搜索
- **Wiki权限**：支持Wiki页面的权限控制

### 7.3 技术实现

- 基于Markdown格式存储Wiki内容
- 实现了Wiki页面的树形结构
- 支持Wiki页面的历史版本管理

### 7.4 使用场景

- 企业内部知识库
- 产品文档管理
- 技术文档中心

## 8. 定时任务支持

### 8.1 功能概述

定时任务功能支持系统自动执行一些周期性的任务，Meblog-Java基于Spring Task实现了定时任务调度。

### 8.2 主要功能点

- **内置定时任务**：包含文章阅读量同步、统计数据更新等内置任务
- **自定义定时任务**：支持开发者添加自定义定时任务
- **任务管理**：支持任务的启用、禁用和参数配置（预留功能）

### 8.3 技术实现

- 基于Spring Task实现
- 支持Cron表达式配置
- 实现了任务的监控和日志记录（预留功能）

### 8.4 使用场景

- 定期同步数据
- 定期生成统计报表
- 定期清理过期数据
- 定期发送通知
